# Kolveniershof

Kolv2Angular is an angular web-app created for the Kolveniershof project. This web-app is an agenda made for the handicapped clients and their mentors.

## Installation

1. Download it
2. Open editor
3. Run "npm i"
4. Run "npm start"
5. Browse to "localhost:4200"
